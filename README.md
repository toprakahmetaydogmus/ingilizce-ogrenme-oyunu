# ULTIMATE+ English (Cloudflare Pages)

## Kurulum (Cloudflare Pages)
1) Bu klasördeki dosyaları **aynen** Cloudflare Pages projenizin `root` dizinine yükleyin.
2) `index.html` otomatik açılır.
3) İlk açılışta internet gerekebilir (ilk cache için). Sonrasında **offline** çalışır.

## Dosyalar
- `index.html` : Tek dosya uygulama (önerilen)
- `single.html` : Aynı uygulamanın alternatif adı
- `database.json`: 700+ kelime, TR anlam + örnek cümle
- `sw.js` + `manifest.webmanifest`: PWA / offline
- `_headers` + `_redirects`: Cloudflare Pages ayarları

## İpuçları
- iOS: Safari > Paylaş > "Ana Ekrana Ekle".
- Android/Chrome: "Uygulamayı yükle" seçeneği çıkar.
